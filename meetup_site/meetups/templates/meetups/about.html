About The Application

My Django project is a simple Meetup Registration System that allows users to view available meetups and register for them. The homepage displays a list of all meetups using the index view, which retrieves all meetups from the database and passes them to the template for display.

When a user clicks on a specific meetup, they are taken to the meetup_details page. This view tries to find the meetup based on the provided slug. If found, it displays the meetup details along with a registration form. If the user submits the form, the app checks if the provided email already exists in the database. If not, a new participant is created and added to the meetup. After successful registration, the user is redirected to a confirmation page.

The confirm_registration view simply displays a confirmation message indicating that the registration was successful. If the meetup does not exist, an error message is shown instead.

Overall, this project provides a basic functionality where users can browse meetups, view details, register using a form, and receive confirmation upon successful registration.
